This project performs an exploratory data analysis (EDA) on the Gapminder dataset to examine relationships between population growth, economic development, and health outcomes across countries from 1952 to 2007.

## Key Analyses
- Life expectancy trends across decades
- GDP per capita distribution by continent
- Correlation analysis between GDP, life expectancy, and population
- Country-level comparison: Afghanistan vs Switzerland

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Strong positive relationship between GDP per capita and life expectancy
- Significant regional disparities in economic and health indicators
- Political and economic stability plays a critical role in long-term development
